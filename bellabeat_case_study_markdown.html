Introduction

This is a case study using Bellabeat. Bellabeat is a successful small company, but they have the potential to become a larger player in the global smart device market. Urka Sren, cofounder and Chief Creative Officer of Bellabeat, believes that analyzing smart device fitness data could help unlock new growth opportunities for the company. My role is to analyse the data available and give insightful recommendations to move the company forward. For more info on the case study, you can Download the original Bellabeat case study PDF

Business Task: *Analyse trends in how users use smart devices

*Recommend how Bellabeat can leverage those trends to market a product better

I wasnt able to include insights for the Spring product because there was no hydration data available in the datasets. Based on the data provided, the most relevant products for analysis are the Bellabeat app, Leaf, and Time.

For this case study, Ive chosen to focus on the Leaf product. While I dont have access to stress-related data, Ill be analyzing the available information on activity, sleep, and weight logs.

The activity dataset gives a full overview of user movement, including steps and intensity levels, so additional datasets on those arent needed. The sleep dataset captures user sleep patterns. *The weight log serves as a useful measure of user engagement with the device.

Clean the data

The data is inconsistent. Im going to summarize it.

First, I will summarize sleep to give me a daily overview.

# Step 1: Convert date column
minuteSleep <- minuteSleep %>%
  mutate(date = mdy_hms(date))  # Fix datetime format

# Step 2: Create a new column just for the "day"
minuteSleep <- minuteSleep %>%
  mutate(sleep_day = as_date(date))

# Step 3: Summarize by user + date
daily_sleep_summary <- minuteSleep %>%
  group_by(Id, sleep_day) %>%
  summarise(
    total_minutes_asleep = sum(value, na.rm = TRUE),   # actual sleep
    total_records = n(),                               # how many minute-rows = time in bed
    .groups = "drop"
  ) %>%
  mutate(total_time_in_bed = total_records)  # if each row = 1 min

#Step 4: Clean up column names
daily_sleep_summary <- daily_sleep_summary %>%
  rename(user_id = Id, date = sleep_day)

Merging daily_activity and daily_sleep_summary

Right now, our dataset is not complete enough for analysis. We need to merge the activity and sleep tables to get a full picture. To do that, well run this code:

activity_sleep <- left_join(
  daily_activity,
  daily_sleep_summary,
  by = c("Id" = "user_id", "ActivityDate" = "date")
)

I used a left join because it keeps all the records from activity, even if a day doesnt have any data for sleep. I didnt want to miss out on important activity info and some users (e.g me) dont wear their smartwatches to sleep.

Now that the datasets have been merged, I want to explore whether theres any relationship between an active lifestyle or an inactive one and sleep duration.

My personal theory is that people who live more active lifestyles tend to sleep better because their bodies have been used to capacity and need more rest to recover.

But thats just a hypothesis, and so were turning to the data. Well use correlation analysis to test whether activity levels (like steps taken, calories burned, and time spent sedentary) show any relationship with how much people sleep.

The goal is to help Bellabeat understand how physical activity influences sleep and how the Leaf can support healthier routines through smart wellness tracking.

Lets get into it:

# This helps us check if more general daily movement leads to more sleep.
cor_steps <- cor(activity_sleep$TotalSteps, activity_sleep$total_minutes_asleep, use = "complete.obs")
cor_steps
## [1] -0.1393673

The correlation between steps and sleep is: -0.1393673. This is a very weak negative correlation, so its not really strong enough to be useful on its own. Its basically saying As people take more steps, they very slightly tend to sleep less.

# This helps us check if workout activity leads to more sleep.
cor_active <- cor(activity_sleep$VeryActiveMinutes, activity_sleep$total_minutes_asleep, use = "complete.obs")
cor_active
## [1] -0.1668336

The correlation between VeryActiveMinutes and sleep is: -0.1668336. This is also a very weak negative correlation and has similar interpretation to the steps correlation. Lets keep going

# This helps us check if using more energy throughout the day(not just workouts and steps) leads to more sleep.
cor_calories <- cor(activity_sleep$Calories, activity_sleep$total_minutes_asleep, use = "complete.obs")
cor_calories
## [1] -0.1686864

The correlation between calories burned and sleep is:-0.1686864 This is still a very weak negative correlation

# This helps us check if using more energy throughout the day(not just workouts and steps) leads to less sleep/lower sleep quality.
cor_sedentary <- cor(activity_sleep$SedentaryMinutes, activity_sleep$total_minutes_asleep, use = "complete.obs")
cor_sedentary
## [1] -0.567461

The correlation between sedentary and sleep is: -0.567461 Now, this is a STRONG negative correlation! This tells us that the more sedentary people are, the less they sleep. Thats a statistically significant relationship

Visualisation

Next, we will create 4 scatter plots to show the relationships between: Steps and Sleep VeryActiveMinutes and Sleep Calories and Sleep SedentaryMinutes and Sleep

We will be mapping the level of activity on the X axis and the total minutes asleep on the Y axis. Well also add a trend line so its easier to see the correlation at a glance.

First, we have to load ggplot2. Its the main package for data viz in R.

p1<- ggplot(activity_sleep, aes(x = TotalSteps, y = total_minutes_asleep)) +
  geom_point(alpha = 0.5, color = "#1b3a4c") +  # Plot the points with transparency

  labs(
    title = "Steps vs Sleep Duration",
    x = "Total Steps",
    y = "Total Minutes Asleep"

  theme_minimal()  # Clean layout

There appears to be a very weak negative correlation between the number of steps taken in a day and the duration of sleep. This suggests that increasing steps doesnt strongly impact sleep, at least not in a predictable way.

Sedentary time has a much stronger negative correlation with sleep duration than the other variables. This suggests that users who spend more time being inactive during the day tend to sleep less. Reducing sedentary behavior could potentially improve sleep outcomes.

Recommendations For Leaf

I recommend the following actions for Leaf:

Since high sedentary time strongly correlates with lower sleep duration, the Leaf app should encourage movement throughout the day. Some options are: *Send personalized reminders to stand up and move if users have been sedentary too long.

*Introduce Daily Movement Goals like a daily steps goal.

*The app could reward consistent daily movement streaks with badges, levels, or community shoutouts.

*Encourage users to set achievable activity challenges, especially during the day.

*Market the Leaf as not just a tracker, but a lifestyle guide. Highlight how it supports users with sedentary jobs or busy routines by helping them reclaim healthy habits.

The Bellabeat Leaf can go beyond tracking and actively support healthier habits by turning data into meaningful, personalized action. Reducing sedentary behavior may be the key to unlocking better sleep and overall wellbeing for Leaf users.
